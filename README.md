# 油猴脚本API示例

[English](README-en.md) | [中文](README.md)

## 简介

这里提供了两个脚本，一个用于展示Tampermonkey脚本API的使用方法，另一个用于展示Greasemonkey 4.0+ 脚本API的使用方法。


- user-script-example.js  Tampermonkey API的使用方法示例
- user-script-example-for-gm.js   Greasemonkey 4.0+ API的使用方法示例


## API 参考

https://github.com/examplecode/tampermonkey-api-reference/blob/master/README.md

## 运行脚本

X浏览器的内置的脚本管理器兼容Tampermonkey API和Greasemonkey 4.0 API, 你可以使用你熟悉的脚本API来实现特定的脚本功能。下面是Tampermonkey API的使用方法示例在X浏览器和Chrome浏览器的测试页面。

https://www.xbext.com/docs/user-script-api-example.html
